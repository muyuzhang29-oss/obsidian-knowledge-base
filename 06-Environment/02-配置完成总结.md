## 🎯 下一步操作

### 立即生效

1. **重新加载 Obsidian**（`Ctrl+Shift+R`）
2. **打开主页**（点击左侧的主页按钮）
3. **查看关系图谱**（`Ctrl+G`）

### 手动配置

1. **应用工作布局**:
   - 命令面板 → "管理工作区布局" → "导入"
   - 选择 `workspace-ideal.json`

2. **启用 CSS 片段**:
   - 设置 → 外观 → CSS 片段
   - 确保所有片段已启用

3. **设置默认主页**:
   - 设置 → 核心插件 → 启用 "日记"
   - 设置日记文件夹为根目录
