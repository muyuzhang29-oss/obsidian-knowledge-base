# 代码块配色方案说明

## 当前使用的配色方案

本知识库代码块采用 **Catppuccin Mocha** 配色方案，提供舒适的阅读体验。

### 颜色配置

| 元素 | 颜色 | 色值 | 说明 |
|------|------|------|------|
| 背景色 | 深蓝灰 | `#1e1e2e` | 深色背景，减少眼睛疲劳 |
| 文字颜色 | 浅灰蓝 | `#cdd6f4` | 主要代码文字颜色 |
| 行内代码 | 靛蓝色 | `#6366f1` (亮) / `#89b4fa` (暗) | 行内代码标识 |
| 行号 | 灰色 | `#6c7086` | 行号显示颜色 |
| 语言标签 | 蓝色 | `#89b4fa` | 代码块右上角的语言标识 |

### 语法高亮颜色（推荐）

| 语法元素 | 颜色 | 色值 |
|----------|------|------|
| 关键字 | 蓝色 | `#89b4fa` |
| 字符串 | 绿色 | `#a6e3a1` |
| 数字 | 橙色 | `#fab387` |
| 注释 | 灰色 | `#6c7086` |
| 函数名 | 青色 | `#89dceb` |

## 如何修改配色方案

### 方法一：修改 CSS 变量

在 `01-知识库美化.css` 文件的 `:root` 部分修改颜色变量：

```css
:root {
  /* 代码块颜色变量 */
  --kb-code-bg: #1e1e2e;
  --kb-code-text: #cdd6f4;
  --kb-code-keyword: #89b4fa;
  --kb-code-string: #a6e3a1;
  --kb-code-number: #fab387;
  --kb-code-comment: #6c7086;
}
```

### 方法二：直接修改代码块样式

在 `01-知识库美化.css` 中找到 `5. 代码块样式` 部分，修改对应颜色值。

## 常见问题

### Q: 为什么代码块显示红色？

**A:** 红色通常由以下原因导致：
1. CSS 中行内代码颜色设置为红色（已修复）
2. 语法高亮主题配置问题
3. 浏览器缓存未刷新

**解决方法：**
- 重启 Obsidian
- 重新加载 CSS 片段
- 清除浏览器缓存（如果使用 Web 版本）

### Q: 如何切换到亮色代码块？

**A:** 将代码块背景色改为浅色：

```css
.markdown-preview-view pre {
  background: #f5f5f5;
}

.markdown-preview-view pre code {
  background: #f5f5f5;
  color: #1e1e2e;
}
```

### Q: 如何调整代码字体大小？

**A:** 修改 `font-size` 属性：

```css
.markdown-preview-view pre code {
  font-size: 0.9em; /* 增大字体 */
}
```
