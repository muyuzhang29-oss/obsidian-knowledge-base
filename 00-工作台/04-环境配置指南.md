## 🚀 应用配置步骤

### 1️⃣ 启用 CSS 片段

1. 打开 Obsidian 设置（`Ctrl+,`）
2. 进入 **外观** → **CSS 片段**
3. 确保以下片段已启用：
   - ✅ `knowledge-base-pro`
   - ✅ `file-explorer-style`
   - ✅ `tag-style`
   - ✅ `folder-color-boxes`
   - ✅ `dashboard-style`

### 2️⃣ 应用工作布局

> [!note] 说明
> Obsidian 的"管理工作区布局"界面只显示**已保存的布局**和**保存按钮**，没有直接的导入功能。

#### 操作步骤

**第一步：保存当前布局（可选）**
1. 打开命令面板（`Ctrl+P`）
2. 输入 **"管理工作区布局"**
3. 点击 **"保存"** → 输入名称如 "当前布局"（备份用）

**第二步：应用理想布局**
1. **关闭 Obsidian**
2. 打开文件夹 `.obsidian/`
3. 将 `workspace-ideal.json` **重命名**为 `workspaces.json`
   - 如果 `workspaces.json` 已存在，先备份再替换
4. **重新打开 Obsidian**
5. 理想布局自动生效（HOME.md 和索引页面已配置好）

**第三步：保存为命名布局（推荐）**
1. 打开命令面板 → **"管理工作区布局"**
2. 点击 **"保存"** → 输入名称 **"理想布局"**
3. 以后可以随时切换回此布局

### 3️⃣ 配置关系图谱

1. 打开关系图谱（`Ctrl+G`）
2. 点击右上角的设置图标
3. 图谱配置已自动应用（`graph.json`）
4. 可根据需要调整颜色分组

### 4️⃣ 设置主页

1. 打开设置 → **核心插件**
2. 确保 **"日记"** 已启用
3. 设置 **"日记文件夹"** 为根目录
4. 设置 **"日记模板"** 为空

## 📝 自定义配置

### 修改文件夹颜色

编辑 `.obsidian/snippets/folder-color-boxes.css` 文件：

```css
/* 修改 01-SV语法 的颜色 */
.nav-folder-title-content[data-path="01-SV语法"] {
  background: linear-gradient(135deg, #fce7f3, #fbcfe8);  /* 背景渐变 */
  color: #9d174d !important;  /* 文字颜色 */
  border-left: 4px solid #ec4899;  /* 左边框颜色 */
}
```

### 修改仪表盘样式

编辑 `.obsidian/snippets/dashboard-style.css` 文件：

```css
/* 修改统计卡片样式 */
.dashboard .callout[data-callout="info"] {
  background: linear-gradient(135deg, #dbeafe, #bfdbfe) !important;
  border: 1px solid #93c5fd !important;
}
```

### 修改关系图谱颜色

编辑 `.obsidian/graph.json` 文件：

```json
{
  "colorGroups": [
    {
      "query": "path:01-SV语法",
      "color": {
        "a": 1,
        "rgb": 14755069  /* 颜色值（十进制） */
      }
    }
  ]
}
```

## ❓ 常见问题

### Q: 文件夹颜色没有显示？

A: 确保 `folder-color-boxes.css` 片段已启用，并重新加载 Obsidian。

### Q: 仪表盘样式没有生效？

A: 确保 `dashboard-style.css` 片段已启用，并在 HOME.md 的 frontmatter 中添加 `cssclass: dashboard`。

### Q: 关系图谱颜色没有变化？

A: 确保 `graph.json` 文件已正确保存，并重新打开关系图谱。

### Q: 工作布局没有应用？

A: 确保已导入 `workspace-ideal.json` 文件，并选择该布局。
