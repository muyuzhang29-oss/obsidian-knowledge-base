# 🎨 CSS美化样式安装说明

## 已完成的步骤

✅ CSS文件已复制到 `.obsidian/snippets/` 目录

## 你需要做的步骤

1. 打开 Obsidian
2. 点击左下角齿轮图标 ⚙️，打开设置
3. 在左侧菜单选择 "外观"
4. 滚动到 "CSS 代码片段" 部分
5. 点击右侧的刷新按钮 🔄
6. 找到 "01-知识库美化"，打开开关 ✅
7. 关闭设置，美化效果立即生效！

## 美化效果

启用后你将看到：
- 🎨 美观的深色主题
- 📝 清晰的标题层级（H1底部粗线、H2左侧竖线）
- 📦 彩色的Callout引用块（7种颜色）
- 📊 现代化的表格样式（斑马纹、悬停高亮）
- ✨ 平滑的动画效果
- 📱 响应式设计（支持手机和平板）
- 🖨️ 打印友好
- 🏷️ 标签自动着色（UVM蓝色、SV紫色、AXI绿色等）

## 自定义修改

如需修改颜色或样式，编辑：
`.obsidian/snippets/01-知识库美化.css`

参考 `00-工作台/07-样式使用说明.md` 了解如何自定义。
