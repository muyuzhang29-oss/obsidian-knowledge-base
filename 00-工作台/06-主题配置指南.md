# Obsidian 主题配置指南

> 本指南为数字验证工程师量身定制，兼顾代码阅读、波形查看、文档编写等场景的视觉需求。

---

## 一、推荐主题

### 1.1 Primary（首选推荐）

- **适用场景**：日常开发文档、UVM 代码笔记、协议规范整理
- **特点**：界面简洁、层级分明、对代码块渲染友好
- **安装路径**：`设置 → 外观 → 主题 → 浏览 → 搜索 "Primary"`
- **推荐配色**：Deep（深色）模式下使用 "Obsidian" 色板

### 1.2 Things

- **适用场景**：任务管理、项目跟踪、待办清单
- **特点**：受 Things 3 启发，卡片式布局，任务状态一目了然
- **安装路径**：`设置 → 外观 → 主题 → 浏览 → 搜索 "Things"`
- **推荐配合**：与 Tasks 插件、Checklist 插件搭配使用效果最佳

### 1.3 Minimal（当前推荐使用）

- **适用场景**：代码密集型笔记、长文档阅读、技术写作
- **特点**：极度精简、留白充足、代码块高亮清晰
- **安装路径**：`设置 → 外观 → 主题 → 浏览 → 搜索 "Minimal"`
- **推荐配合**：已安装 `obsidian-minimal-settings` 和 `obsidian-style-settings` 插件，可深度定制
- **优势**：对 SystemVerilog/UVM 语法高亮支持好，阅读长篇 RTL 代码不疲劳

### 1.4 AnuPpuccin

- **适用场景**：个性化定制、多配色切换、视觉偏好驱动
- **特点**：Catppuccin 配色体系、支持多种强调色、渐变效果丰富
- **安装路径**：`设置 → 外观 → 主题 → 浏览 → 搜索 "AnuPpuccin"`
- **推荐配色**：Mocha（深色）/ Latte（浅色）

## 二、颜色方案配置

### 2.1 深色模式（推荐用于长时间编码/阅读）

```
设置 → 外观 → 深色模式：开启
```

**推荐参数**：

| 配置项 | 推荐值 | 说明 |
|--------|--------|------|
| 背景色 | `#1e1e2e` | 深蓝灰，减少眼部疲劳 |
| 正文色 | `#cdd6f4` | 浅灰蓝，对比度适中 |
| 强调色 | `#89b4fa` | 柔和蓝，适合高亮关键信号 |
| 代码块背景 | `#181825` | 比正文更深，代码区域视觉隔离 |
| 链接色 | `#89dceb` | 青色，便于识别超链接 |
| 标题色 | `#f5c2e7` | 粉紫色，层级区分明显 |

### 2.2 浅色模式（适合打印/演示）

```
设置 → 外面 → 深色模式：关闭
```

**推荐参数**：

| 配置项 | 推荐值 | 说明 |
|--------|--------|------|
| 背景色 | `#eff1f5` | 柔和白，不刺眼 |
| 正文色 | `#4c4f69` | 深灰，阅读舒适 |
| 强调色 | `#1e66f5` | 标准蓝，清晰醒目 |
| 代码块背景 | `#e6e9ef` | 浅灰，代码区域区分 |
| 链接色 | `#209fb5` | 青蓝，链接可辨识 |

### 2.3 快速切换方案

通过 Style Settings 插件预设多套配色：

1. **工作模式**（深色）：日常编码和笔记
2. **演示模式**（浅色）：会议分享、报告展示
3. **专注模式**（深色高对比）：阅读复杂协议文档

## 三、字体配置

### 3.1 推荐字体组合

#### 方案 A：经典技术风格（推荐）

```css
/* 正文/标题字体 */
--font-text: "LXGW WenKai", "Noto Sans SC", sans-serif;

/* 等宽字体（代码块） */
--font-monospace: "JetBrains Mono", "Fira Code", "Cascadia Code", monospace;

/* 界面字体（侧边栏/标签） */
--font-interface: "Inter", "Microsoft YaHei", sans-serif;
```

#### 方案 B：极简现代风格

```css
--font-text: "HarmonyOS Sans SC", "PingFang SC", sans-serif;
--font-monospace: "Source Code Pro", "Consolas", monospace;
--font-interface: "HarmonyOS Sans SC", sans-serif;
```

#### 方案 C：高可读性风格（适合长时间阅读）

```css
--font-text: "Source Han Serif SC", "Noto Serif SC", serif;
--font-monospace: "IBM Plex Mono", "JetBrains Mono", monospace;
--font-interface: "IBM Plex Sans", "Microsoft YaHei", sans-serif;
```

### 3.2 字体大小与行高

```css
/* 推荐参数 */
--font-text-size: 16px;          /* 正文字号 */
--font-text-line-height: 1.8;    /* 行高，代码密集时增大 */
--font-monospace-size: 14px;     /* 代码字号 */
--font-heading-size-scale: 1.3;  /* 标题字号缩放比 */
```

### 3.3 字体安装说明

- **LXGW WenKai**（霞鹜文楷）：开源中文字体，GitHub 下载
- **JetBrains Mono**：JetBrains 凑件字体，连字支持好
- **Cascadia Code**：微软终端字体，Windows 自带或通过 Windows Terminal 安装
- **Fira Code**：Mozilla 系等宽字体，连字效果优秀

## 四、界面布局建议

### 4.1 推荐布局（数字验证工程师专用）

```
┌─────────────────────────────────────────────────────┐
│ 标签栏                                               │
├──────────┬──────────────────────────────┬───────────┤
│          │                              │           │
│ 左侧边栏  │        编辑区/阅读区          │  右侧边栏  │
│ (文件树)  │                              │ (大纲/反链)│
│          │                              │           │
│ 文件列表   │  ┌──────────────────────┐    │  大纲导航  │
│ 搜索      │  │      正文内容         │    │  反向链接  │
│ 标签      │  │                      │    │  标签列表  │
│          │  │  代码块 / 表格 / 图片   │    │           │
│          │  │                      │    │           │
│          │  └──────────────────────┘    │           │
│          │                              │           │
├──────────┴──────────────────────────────┴───────────┤
│ 状态栏                                               │
└─────────────────────────────────────────────────────┘
```

### 4.2 布局配置步骤

1. **开启左侧面板**：`Ctrl/Cmd + \` 显示文件资源管理器
2. **开启右侧面板**：`Ctrl/Cmd + Shift + \` 显示大纲和反向链接
3. **设置默认视图**：`设置 → 编辑器 → 默认编辑模式 → 阅读模式`（适合文档查阅）
4. **标签页布局**：`设置 → 外观 → 窗口 → 显示标签页标题栏`

### 4.3 工作区预设（使用 Workspaces 插件）

| 工作区名称 | 用途 | 布局特点 |
|-----------|------|---------|
| **代码阅读** | SystemVerilog/UVM 源码分析 | 左侧文件树 + 中间代码 + 右侧大纲 |
| **文档编写** | 技术文档、验证计划 | 左侧大纲 + 中间编辑器 + 右侧预览 |
| **项目管理** | 任务跟踪、进度管理 | 左侧看板 + 中间日历 + 右侧任务列表 |
| **知识浏览** | 快速查阅笔记 | 全屏阅读模式，隐藏侧边栏 |
| **仪表盘** | 工作台总览 | 居中卡片布局，统计面板 |


- 高频操作建议使用单手可按的组合
- 通过 Commander 插件可为常用命令添加工具栏按钮

## 九、常见问题

**Q: 主题切换后样式异常怎么办？**
A: 清除 `设置 → 外观 → CSS 代码片段` 中可能冲突的 snippet，重启 Obsidian。

**Q: 代码块字体不生效？**
A: 确认字体已正确安装，检查 `--font-monospace` 变量是否被 snippet 覆盖。

**Q: 如何备份主题配置？**
A: 将 `.obsidian` 目录纳入 Git 版本控制，或使用 Remotely Save 插件同步。

**Q: 深色模式下代码高亮颜色太暗？**
A: 通过 Style Settings 调整代码块背景色对比度，或使用 Code Block Customizer 插件单独配置。
