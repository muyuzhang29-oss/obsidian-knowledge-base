# Linux 常用命令

> 数字IC验证工程师必备的Linux操作

## 📁 文件操作

### 浏览文件/目录

```bash
# 列出目录内容
ls -la                    # 详细列表
ls -lh                    # 人性化大小
ls -lt                    # 按时间排序
ls -S                     # 按大小排序
ls -a                     # 显示隐藏文件

# 切换目录
cd ~                      # 回家目录
cd -                      # 上一个目录
cd ..                     # 上级目录
cd /path/to/dir           # 绝对路径

# 查看当前目录
pwd                       # 打印工作目录
```

### 文件操作

```bash
# 复制
cp file1 file2           # 复制文件
cp -r dir1 dir2           # 复制目录
cp file1 dir/             # 复制到目录

# 移动/重命名
mv file1 file2           # 重命名
mv file1 dir/            # 移动到目录

# 删除
rm file                  # 删除文件
rm -rf dir               # 强制删除目录
rm -i file               # 交互式删除

# 创建
touch file               # 创建空文件
mkdir -p dir/subdir      # 创建目录树
mkdir -m 755 dir         # 指定权限

# 查找
find . -name "*.v"       # 查找SV文件
find . -name "*.sv" -mtime -7  # 7天内修改的
find . -size +100M        # 大于100M的文件
grep -r "pattern" .       # 递归搜索
grep -n "pattern" file    # 显示行号
```

## 🔧 文件查看

### 文本查看

```bash
# 查看文件
cat file                  # 全部显示
cat -n file              # 显示行号
less file                 # 分页查看
head -20 file            # 前20行
tail -20 file            # 后20行
tail -f log.txt          # 实时跟踪

# 统计
wc -l file               # 行数
wc -w file               # 单词数
wc -c file               # 字符数

# 比较
diff file1 file2         # 比较文件
vimdiff file1 file2      # vim比较
```

## 📦 压缩/解压

```bash
# tar压缩
tar -cvf archive.tar dir/       # 压缩
tar -xvf archive.tar              # 解压
tar -czvf archive.tar.gz dir/     # gzip压缩
tar -xzvf archive.tar.gz          # gzip解压

# zip
zip -r archive.zip dir/           # 压缩
unzip archive.zip                 # 解压
```

## 🔐 权限管理

```bash
# 权限格式: rwx rwx rwx (owner group other)
# r=4, w=2, x=1

chmod 755 file                   # rwxr-xr-x
chmod +x script.sh               # 添加执行权限
chmod -R 644 *.txt              # 递归设置

chown user:group file            # 更改所有者
chgrp group file                 # 更改组
```

## ⚙️ 系统命令

```bash
# 进程
ps aux                          # 查看进程
ps -ef | grep process           # 查找进程
kill -9 PID                     # 强制终止
top                             # 任务管理器
htop                            # 增强top

# 内存/磁盘
free -h                         # 内存
df -h                          # 磁盘
du -sh dir                     # 目录大小

# 其他
date                            # 日期时间
whoami                          # 用户名
hostname                        # 主机名
history                         # 命令历史
alias ll='ls -la'              # 设置别名
```

## 🔧 仿真相关

```bash
# 编译仿真
make                           # 执行Makefile
make clean                     # 清理
make compile                   # 编译

# 文件操作
cat filelist.f                # 查看文件列表
grep -n "module" *.v           # 查找模块

# 波形
verdi -sv -f filelist.f &     # 打开Verdi
dve -vpd wave.vpd &            # 打开DVE

# 日志
grep -i error sim.log         # 查找错误
grep -i warning sim.log        # 查找警告
tail -f sim.log                # 实时跟踪
```

## 📝 文本处理

```bash
# 管道与重定向
command > output.txt            # 输出到文件
command >> output.txt           # 追加
command 2>&1                   # 错误合并输出
command1 | command2            # 管道

# sed流编辑器
sed -i 's/old/new/g' file     # 替换
sed -n '10,20p' file          # 10-20行

# awk
awk '{print $1}' file          # 第一列
awk -F: '{print $1}' file      # 指定分隔符

# 排序去重
sort file | uniq              # 去重
sort -k2 file                 # 按第2列排序
```

## 🚀 快捷技巧

```bash
# 命令历史
!!                          # 上一条命令
!n                          # 第n条命令
!string                     # 最近以string开头的
Ctrl+R                      # 反向搜索

# 自动补全
Tab                         # 自动补全
Tab Tab                     # 显示所有选项

# 光标移动
Ctrl+A                      # 行首
Ctrl+E                      # 行尾
Ctrl+U                      # 删除整行
Ctrl+K                      # 删除到行尾
Ctrl+L                      # 清屏
```
