# GVim 快捷键

> 高效编辑代码的必备技能

## 📁 基本操作

### 模式切换

| 按键 | 说明 |
|------|------|
| `i` | 进入插入模式 |
| `a` | 在光标后插入 |
| `A` | 在行尾插入 |
| `o` | 下一行插入 |
| `O` | 上一行插入 |
| `Esc` | 返回普通模式 |
| `:` | 进入命令模式 |
| `v` | 可视模式 |
| `V` | 行可视模式 |
| `Ctrl+v` | 块可视模式 |

### 退出保存

```vim
:w                          " 保存
:wq                         " 保存退出
:x                          " 保存退出(更快)
:q!                         " 强制退出
:qa!                        " 强制退出所有
ZZ                          " 保存退出
ZQ                          " 不保存退出
```

## ✏️ 编辑操作

### 插入与删除

```vim
i                          " 插入
a                          " 追加
o                          " 下一行
O                          " 上一行

x                          " 删除光标字符
dd                         " 删除整行
ndd                        " 删除n行
dw                         " 删除单词
d$                         " 删除到行尾
d^                         " 删除到行首

yy                         " 复制整行
nyy                        " 复制n行
yw                         " 复制单词
y$                         " 复制到行尾

p                          " 粘贴
np                         " 粘贴n次

u                          " 撤销
Ctrl+r                     " 重做
.                          " 重复上次操作
```

### 修改与替换

```vim
r                          " 替换单个字符
R                          " 进入替换模式
~                          " 大小写切换
cc                         " 替换整行
s                          " 删除并插入

:1,10s/old/new/g           " 1-10行替换
:%s/old/new/g               " 全局替换
:%s/old/new/gc              " 确认替换
```

## 🔍 搜索跳转

### 搜索

```vim
/pattern                    " 向下搜索
?pattern                    " 向上搜索
n                          " 下一个匹配
N                          " 上一个匹配
*                          " 搜索光标单词
#                          " 反向搜索单词

:set hlsearch              " 高亮搜索
:set incsearch             " 增量搜索
:nohlsearch                " 清除高亮
:set ignorecase            " 忽略大小写
```

### 跳转

```vim
gg                         " 文件开头
G                          " 文件结尾
nG                         " 跳转到第n行
Ctrl+o                     " 跳转历史-返回
Ctrl+i                     " 跳转历史-前进

%                          " 匹配括号跳转
{                          " 上一段
}                          " 下一段
[[                         " 上一函数
]]                         " 下一函数
```

## 📝 代码编辑

### 缩进

```vim
>>                         " 增加缩进
<<                         " 减少缩进
==                         " 自动缩进行
n>>                        " n行增加缩进

:set autoindent            " 自动缩进
:set smartindent           " 智能缩进
:set expandtab             " Tab转空格
:set ts=4                  " Tab宽度4
:set sw=4                  " 缩进宽度4
```

### 注释

```vim
" 单行注释
Ctrl+v → 选择行 → I → // → Esc

" 取消注释
Ctrl+v → 选择 // → d

" 批量注释 (Linux)
:1,10s/^/\/\/ /            " 1-10行加注释
:1,10s/^\/\/ //            " 1-10行去注释
```

### 代码折叠

```vim
zf                         " 创建折叠
zo                          " 打开折叠
zc                          " 关闭折叠
zR                          " 打开所有
zM                          " 关闭所有
za                          " 切换折叠
```

## 🔧 高级技巧

### 寄存器

```vim
"ayy                        " 复制到寄存器a
"ap                         " 从寄存器a粘贴
:reg                        " 查看所有寄存器

"0                          " 最近复制
"1-"9                       " 删除历史
"-                          " 小删除

"                            " 匿名寄存器
_                            " 黑洞寄存器
```

### 宏

```vim
qa                          " 开始录制宏a
...操作...
q                           " 停止录制
@a                          " 执行宏a
@@                          " 重复上次宏
10@a                        " 执行10次
```

### 补全

```vim
Ctrl+x Ctrl+o              " Omni补全
Ctrl+x Ctrl+l              " 行补全

```

## 🎨 快捷键映射

```vim
" .vimrc 示例
let mapleader = ","

" 快速保存
nnoremap <leader>w :w<CR>

" 快速退出
nnoremap <leader>q :q<CR>

" 切换行号
nnoremap <leader>n :set nu!<CR>

" 快速搜索
nnoremap <leader>/ :nohlsearch<CR>

" 格式化代码
nnoremap <leader>f =%

" 复制全文
nnoremap <leader>y ggVGy
```

## 📝 SystemVerilog 模板

```vim
" .vim/ftplugin/systemverilog.vim

" 自动插入模板
autocmd BufNewFile *.sv call SVTemplate()

function! SVTemplate()
    call setline(1, '// =======================================')
    call append(1, '// Module: ')
    call append(2, '// Author: ')
    call append(3, '// Date: ' . strftime('%Y-%m-%d'))
    call append(4, '// =======================================')
    call append(5, '')
    call append(6, 'class ' . expand('%:r:t') . ' extends uvm_')
    normal G
endfunction
```
