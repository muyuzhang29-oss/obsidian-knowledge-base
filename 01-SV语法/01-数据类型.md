# 01-数据类型

## 数据类型概述

SystemVerilog 引入了两种数据类型系统：
- **4 值逻辑**：`logic`、`integer`、`byte`、`shortint`、`int`、`longint`
- **2 值逻辑**：`bit`、`real`、`shortreal`

## 4 值逻辑 vs 2 值逻辑

| 类型 | 值域 | 用途 |
|------|------|------|
| `logic` | 0, 1, X, Z | RTL 设计、端口信号 |
| `bit` | 0, 1 | 验证环境内部信号 |
| `integer` | 0, 1, X, Z, 有符号 | 循环计数、计算 |
| `byte` | -128 ~ 127 | 字符处理 |

```verilog
logic       [7:0]  data_4state;  // 可以是 X 或 Z
bit       [7:0]  data_2state;  // 只有 0 或 1
```

## 有符号与无符号

```verilog
logic signed   [7:0] signed_data;   // -128 ~ 127
logic unsigned [7:0] unsigned_data; // 0 ~ 255

// 隐式转换规则
bit [15:0] a = 8'hFF;     // a = 0x00FF (零扩展)
logic signed [15:0] b = 8'hFF;  // b = 0xFFFF (符号扩展)
```

## 定宽数组

```verilog
// 一维数组
bit [7:0] array[4];           // 4 个 8bit 元素的数组
bit [7:0] mem [0:255];        // 256 深度的内存

// 多维数组
bit [3:0] matrix [2:0][3:0];  // 3x4 的 4bit 数组

// 紧凑型数组声明（ packed array）
bit [3:0][7:0] packed;        // 32bit 紧凑数组
```

### 数组初始化

```verilog
int arr1[4] = '{0, 1, 2, 3};      // 完整初始化
int arr2[4] = '{default: -1};    // 所有元素设为 -1
int arr3[4] = '{0, 1, default:8}; // 0,1,8,8
```

## 动态数组

```verilog
int dyn_array[];
bit [7:0] data[];

initial begin
    dyn_array = new[10];           // 分配 10 个元素
    dyn_array[0] = 100;
    dyn_array = new[20](dyn_array); // 扩展并复制
end
```

## 关联数组

```verilog
bit [63:0] assoc_array[string];  // string 作为索引
int data[*];                      // 任意整数索引

initial begin
    assoc_array["key1"] = 100;
    assoc_array["key2"] = 200;
    foreach(assoc_array[i])
        $display("%s: %0d", i, assoc_array[i]);
end
```

## 队列

```verilog
int q[$];
bit [31:0] data_q[$];

initial begin
    data_q.push_back(10);    // 队尾插入
    data_q.push_front(5);   // 队首插入
    int val = data_q.pop_back();  // 队尾取出
    val = data_q.pop_front();      // 队首取出
    data_q.delete(0);       // 删除第 0 个元素
end
```

## 字符串

```verilog
string str1 = "Hello";
string str2;
int len;

initial begin
    str2 = {str1, " World"};    // 字符串连接
    len = str1.len();           // 长度
    str2.toupper();             // 转大写
    str2.putc(0, "h");          // 替换字符
    if (str1 == "Hello") begin  // 比较
    end
end
```

## 结构体

```verilog
// 未打包结构体
struct {
    bit [7:0] addr;
    bit [31:0] data;
    bit        valid;
} packet_t;

// 打包结构体（连续存储）
typedef struct packed {
    bit        valid;
    bit [31:0] data;
    bit [7:0]  addr;
} packed_packet_t;
```

## 枚举类型

```verilog
typedef enum logic [2:0] {
    IDLE   = 3'b000,
    START  = 3'b001,
    READ   = 3'b010,
    WRITE  = 3'b011,
    STOP   = 3'b100
} state_t;

state_t state, next_state;
```

## 类型转换

### 静态转换
```verilog
int i;
bit [7:0] b;
b = bit'(i);        // 位宽转换
i = int'(b);        // 整数转换
```

### 动态转换（casting）
```verilog
int i;
longint unsigned u;
u = longint'(i);    // 有符号转无符号
```

### 流操作符
```verilog
bit [7:0] byte_data;
bit [31:0] word;
byte_data = {>>{word}};     // 大端字节序
byte_data = {<<{word}};     // 小端字节序
byte_data = {<<8{word}};   // 逐字节反转
```

## 常见错误

### 1. 数组字面量类型推断
```verilog
// ❌ 可能出错
int arr = '{1, 2, 3};  // 缺少维度

// ✅ 正确
int arr[3] = '{1, 2, 3};
```

### 2. signed/unsigned 混淆
```verilog
logic [7:0] a = 8'hFF;
logic [7:0] b;
b = a + 1;     // 结果 0（无符号溢出）
b = unsigned'(a) + 1;  // 结果 256
```
