# SystemVerilog 入门

> [!abstract] 概述
> SystemVerilog 是 Verilog 的扩展，集成了硬件描述语言（HDL）和硬件验证语言（HVL）特性。它是验证工程师的核心工具，主要用于：

- RTL 设计描述
- 高级验证方法学
- 覆盖率驱动验证
- 断言验证（SVA）

## 基本语法结构

### Module 定义

```systemverilog
module tb_top;
    // 信号声明
    logic        clk;
    logic        rst_n;
    logic [31:0] data_in;
    logic [31:0] data_out;

    // 实例化 DUT
    dut u_dut (
        .clk      (clk),
        .rst_n    (rst_n),
        .data_in  (data_in),
        .data_out (data_out)
    );

    // 时钟生成
    initial begin
        clk = 0;
        forever #5 clk = ~clk;
    end

    // 测试逻辑
    initial begin
        rst_n = 0;
        #20 rst_n = 1;
        // ...
    end
endmodule
```

### 数据类型

| 类型 | 位宽 | 描述 |
|------|------|------|
| `logic` | 1bit | 单比特信号 |
| `bit` | 1bit | 无符号单比特 |
| `byte` | 8bit | 字节类型 |
| `int` | 32bit | 有符号整数 |
| `bit [31:0]` | 32bit | 无符号向量 |

### 过程块

```systemverilog
// 组合逻辑
always_comb begin
    out = a & b;
end

// 时序逻辑
always_ff @(posedge clk) begin
    if (rst_n == 0)
        q <= 0;
    else
        q <= d;
end

// 组合逻辑 always 块（兼容老代码）
always @(*) begin
    // 组合逻辑
end
```

## 与 Verilog 的区别

| 特性 | Verilog | SystemVerilog |
|------|---------|---------------|
| 变量声明 | `reg/wire` | `logic/var` |
| 4 值逻辑 | 支持 | 支持 |
| 2 值逻辑 | 不支持 | 支持 |
| 类和对象 | 不支持 | 支持 |
| 接口 | 不支持 | 支持 |
| 断言 | 基础 | 完整 SVA |

## 常见错误

### 1. 多驱动问题

```systemverilog
// ❌ 错误：多个 always 块驱动同一信号
always @(posedge clk) q <= a;
always @(posedge clk) q <= b;

// ✅ 正确：使用优先级或选择逻辑
always @(posedge clk) begin
    if (sel) q <= a;
    else     q <= b;
end
```

### 2. 悬空信号

```systemverilog
// ❌ 错误：output 未赋值
output [7:0] data_out;

// ✅ 正确：确保所有输出都有赋值
output logic [7:0] data_out;
```
