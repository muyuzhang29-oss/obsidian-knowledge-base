# 02-波形调试技巧

> [!abstract] 概述
> 收录 Verdi 波形调试的常用操作、技巧和方法，帮助高效定位仿真问题。

---

## Verdi 常用操作

### 1. 启动 Verdi

```bash
# 从仿真工具启动
xrun -gui -f filelist.f
vcs -full64 -debug_access+all -kdb -f filelist.f

# 单独启动 Verdi
verdi -ssf waveform.fsdb -sswr signal.rc

# 加载设计数据库
verdi -dbdir simv.daidir
```

### 2. 基本界面操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+W` | 波形窗口 |
| `Ctrl+S` | 源代码窗口 |
| `Ctrl+T` | 层次窗口 |
| `Ctrl+L` | 逻辑窗口 |
| `Ctrl+P` | 进程窗口 |
| `Ctrl+D` | 数据流窗口 |
| `F5` | 运行 |
| `Shift+F5` | 运行到光标 |
| `F9` | 设置断点 |
| `Ctrl+F` | 查找信号 |

### 3. 信号添加与管理

```bash
# 从层次窗口添加信号
# 1. 在层次窗口展开模块
# 2. 选择信号 -> 右键 -> "Add to Wave"

# 从源代码添加信号
# 1. 在源代码窗口选择信号
# 2. 右键 -> "Add to Wave"

# 批量添加信号
# 使用通配符：*data*, *valid*, *ready*

# 信号分组
# 选择多个信号 -> 右键 -> "Group" -> 命名
```

### 4. 波形查看与导航

| 操作 | 方法 |
|------|------|
| 缩放 | 鼠标滚轮 或 `+`/`-` |
| 平移 | 鼠标中键拖动 或 `←`/`→` |
| 适应窗口 | `F` 或 `View -> Fit` |
| 跳转到时间 | `Ctrl+G` -> 输入时间 |
| 跳转到标记 | `Ctrl+M` -> 选择标记 |
| 标记当前时间 | `M` 或 `Edit -> Marker` |
| 测量时间差 | 选择两个标记 -> 右键 -> "Measure" |

## 波形查看技巧

### 1. 信号显示格式

```
# 右键信号 -> "Radix" 选择显示格式
Hex     - 十六进制
Decimal - 十进制
Binary  - 二进制
Octal   - 八进制
ASCII   - ASCII 字符
Float   - 浮点数
Enum    - 枚举类型
```

### 2. 信号分组管理

```
# 创建信号组
1. 选择相关信号
2. 右键 -> "Group"
3. 命名组（如：AXI_BUS, CONTROL, STATUS）

# 常用分组建议
├── AXI_BUS
│   ├── awvalid, awready, awaddr
│   ├── wvalid, wready, wdata
│   ├── bvalid, bready, bresp
│   ├── arvalid, arready, araddr
│   └── rvalid, rready, rdata, rresp
├── CONTROL
│   ├── clk, rst_n
│   ├── en, valid, ready
│   └── state, next_state
└── STATUS
    ├── error, done
    └── count, status
```

### 3. 总线信号查看

```bash
# 创建总线信号
1. 选择多个位信号（如 data[7:0]）
2. 右键 -> "Combine" -> "Bus"
3. 设置总线名称和位宽

# 或使用表达式
# 在 "Add Signal" 对话框输入表达式
# 如：{data[7:0], addr[3:0]}
```

### 4. 信号搜索与过滤

```bash
# 按名称搜索
Ctrl+F -> 输入信号名（支持通配符 * ?）

# 按层次过滤
层次窗口 -> 右键 -> "Filter" -> 输入过滤条件

# 按类型过滤
层次窗口 -> 右键 -> "Filter by Type"
- Registers
- Wires
- Ports
- Parameters
```

### 5. 波形比较

```bash
# 加载多个波形文件
1. File -> "Open Database"
2. 选择不同的 FSDB 文件
3. 在波形窗口并排查看

# 波形差异高亮
1. 选择两个波形
2. 右键 -> "Compare"
3. 差异部分会高亮显示
```

### 6. 数据流追踪

```bash
# 启动数据流窗口
1. 选择信号 -> 右键 -> "Trace" -> "Driver/Load"
2. 或使用快捷键 Ctrl+D

# 数据流操作
- 前向追踪：选择驱动信号 -> "Trace Forward"
- 后向追踪：选择负载信号 -> "Trace Backward"
- 全路径追踪：选择信号 -> "Trace All"
```

## 断点设置方法

### 1. 源代码断点

```bash
# 在源代码窗口设置断点
1. 打开源文件（Ctrl+S）
2. 点击行号左侧空白区域
3. 出现红色圆点表示断点已设置

# 条件断点
1. 右键断点 -> "Properties"
2. 设置条件表达式
3. 如：data == 32'hFFFF_FFFF
```

### 2. 信号断点

```bash
# 信号值变化断点
1. 在波形窗口选择信号
2. 右键 -> "Breakpoint" -> "Value Change"
3. 仿真会在信号值变化时暂停

# 信号特定值断点
1. 选择信号 -> 右键 -> "Breakpoint" -> "Value"
2. 设置目标值
3. 如：valid == 1'b1
```

### 3. 断点管理

```bash
# 断点窗口
View -> "Breakpoints" 或 Ctrl+B

# 断点操作
- 启用/禁用：点击断点左侧复选框
- 删除：选择断点 -> Delete
- 编辑：双击断点 -> 修改属性
- 条件设置：右键 -> "Properties"
```

### 4. 断点表达式

```systemverilog
// 常用断点表达式

// 信号等于特定值
data == 32'h1234_5678

// 信号范围
addr >= 32'h1000 && addr <= 32'h1FFF

// 信号变化
$rose(valid)
$fell(ready)
$changed(state)

// 多信号组合
valid && ready && (data == 32'hFFFF)

// 时间条件
$time > 1000
```

### 5. 断点操作快捷键

| 快捷键 | 功能 |
|--------|------|
| `F9` | 设置/取消断点 |
| `F5` | 运行到下一个断点 |
| `Shift+F5` | 运行到光标位置 |
| `F10` | 单步（Step Over） |
| `F11` | 单步进入（Step Into） |
| `Shift+F11` | 单步跳出（Step Out） |

## 常见调试场景

### 场景 1：信号值不正确

```
问题：某个信号值不符合预期
排查步骤：
1. 添加信号到波形窗口
2. 追溯信号驱动源
3. 检查驱动逻辑
4. 验证输入条件
5. 检查时序关系
```

### 场景 2：状态机卡死

```
问题：状态机卡在某个状态
排查步骤：
1. 添加状态信号到波形
2. 设置状态信号断点
3. 检查状态转换条件
4. 验证输入信号
5. 检查时序约束
```

### 场景 3：时序违规

```
问题：建立/保持时间违规
排查步骤：
1. 查看时钟和数据信号
2. 测量时钟到数据延迟
3. 检查组合逻辑延迟
4. 分析时序路径
5. 优化关键路径
```

### 场景 4：协议违规

```
问题：总线协议违规
排查步骤：
1. 添加总线信号
2. 使用协议检查器
3. 查看协议违规点
4. 分析违规原因
5. 修复协议问题
```

## 波形文件管理

### 1. FSDB 文件生成

```bash
# VCS 生成 FSDB
vcs -full64 -sverilog \
    -debug_access+all \
    -kdb \
    -f filelist.f

# xrun 生成 FSDB
xrun -sv -uvm \
    -access +rwc \
    -f filelist.f

# 在代码中调用
initial begin
    $fsdbDumpfile("waveform.fsdb");
    $fsdbDumpvars(0, tb_top);
end
```

### 2. 波形配置保存

```bash
# 保存波形配置
File -> "Save Signal" -> 保存为 .rc 文件

# 加载波形配置
File -> "Load Signal" -> 选择 .rc 文件

# 配置文件位置
~/.novas/verdi.rc  （全局配置）
./verdi.rc         （项目配置）
```

### 3. 波形截图与导出

```bash
# 截图
File -> "Export" -> "Image"
支持格式：PNG, JPEG, BMP, TIFF

# 导出数据
File -> "Export" -> "Data"
支持格式：CSV, VCD, FSDB
```
