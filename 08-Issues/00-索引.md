## 按标签索引

### #Issues - 问题追踪

```dataview
TABLE file.folder AS "分类", file.tags AS "标签"
FROM "09-Issues"
WHERE contains(file.tags, "Issues")
SORT file.name
```

### #Compilation - 编译问题

```dataview
TABLE file.folder AS "分类", file.tags AS "标签"
FROM "09-Issues"
WHERE contains(file.tags, "Compilation")
SORT file.name
```

### #Debug - 调试问题

```dataview
TABLE file.folder AS "分类", file.tags AS "标签"
FROM "09-Issues"
WHERE contains(file.tags, "Debug") OR contains(file.tags, "调试")
SORT file.name
```

### #Coverage - 覆盖率问题

