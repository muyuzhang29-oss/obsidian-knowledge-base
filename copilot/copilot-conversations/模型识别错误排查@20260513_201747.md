*   **Kanban (看板)**: 右侧 "Focus On Today" 部分可能使用了看板插件，或者是 Dataview 做的任务列表。

截图中的风格非常像 **Minimal** 主题或者 **Things** 主题的深色模式，配合了自定义的 CSS 颜色。

*   **推荐主题**: `Minimal` (最灵活) 或 `Primary`。
*   **配色方案**: 使用了深色背景搭配粉色/紫色的高亮色（Accent Color）。

### 4. 布局结构 (Layout)

1.  **CSS Snippets (CSS 代码片段)**: 自己写或找现成的 CSS 代码，将页面分成左、中、右三栏。
2.  **Omnisearch / Homepage 插件**: 使用插件自带的布局功能。

### 5. 侧边栏 (Right Sidebar)

*   **Calendar 插件**: 放在右侧边栏。
*   **Tasks 插件** 或 **Dataview**: 用于显示 "Focus On Today" 的待办事项。
